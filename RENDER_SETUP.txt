HOW TO GET PREREQUISITES:
=========================

MONGODB_URI - MongoDB Atlas Setup:
1. Go to https://www.mongodb.com/cloud/atlas
2. Create free account
3. Create Organization and Project
4. Create M0 Cluster (free tier)
5. Create Database User with username and password
6. Whitelist your IP (add 0.0.0.0/0 for open access)
7. Click "Connect" → "Drivers" → Copy connection string
8. Replace <username> and <password> with your credentials
9. Replace "myFirstDatabase" with "indian-stock-basket"
10. Example: mongodb+srv://myuser:mypass@cluster0.xyz.mongodb.net/indian-stock-basket

EMAIL_PASSWORD - Gmail App Password:
1. Enable 2-Factor Authentication on your Gmail account
   https://myaccount.google.com/security
2. Go to https://myaccount.google.com/apppasswords
3. Select "Mail" and "Windows Computer"
4. Google will generate a 16-character password
5. Copy this password (remove spaces)
6. Use this as EMAIL_PASSWORD (NOT your regular Gmail password)

JWT_SECRET - Generate Random:
1. Simple option: Use "your_super_secret_key_12345"
2. Or generate: Open any online random string generator
3. Copy a 32-character random string

TROUBLESHOOTING:
================
Build failed?
- Check the build logs on Render dashboard
- Verify all environment variables are set
- Verify Root Directory is "backend"
- Check that build command is: cd backend && npm install

API not responding?
- Check Render logs (Logs tab on dashboard)
- Verify MongoDB connection string is correct
- Verify all environment variables are set

Still stuck?
- Render support: https://render.com/docs
